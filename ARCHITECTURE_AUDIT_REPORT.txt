SECTION 13: TOP 20 CRITICAL GAPS DEBARRED FROM STA4CAD-LEVEL DELIVERABLES
================================================================================
These gaps must be resolved for the software to fully match professional STA4CAD-tier engineering capabilities:

1. Bidirectional Drawing Sync: Changes made within the interactive detailing canvas do not update the 3D analytical model.
2. 3D Sheet Viewport Render: Lacks a 3D wireframe preview capability on paper space layouts.
3. Advanced CAD Hatching Hatch: DXF output patterns (ANSI32, concrete aggregate) are simulated using dense line strings rather than native AutoCAD pattern fills.
4. Long-Term Slab Deflection Curves: Concrete slab deflection (long-term creep and shrinkage) curves are not drawn visually on sectional cuts.
5. Parametric Pile Cap Detailing: Pile cap reinforcement and strap tie layouts are handled via manual schedules rather than active generative drawings.
6. Multi-Viewport Layout Optimizations: Sheet layout viewport positioning relies on rigid rules rather than responsive boundaries.
7. Retaining Wall Detailing Library: Lacks distinct detailing templates for complex concrete retaining structures (treated as column-like segments).
8. Circular Spiral Column Ties: Detailing calculations support standard circular ties but lack helical spiral wrapping detailing layouts.
9. Parametric Concrete Stairs Detailing: stair details (sloped flights, landings, steps, reinforcement, starter dowels) are handled via notes, not parametric detailing vectors.
10. Splayed Footprint Slab Details: Triangular and irregular polygonal slab layouts lack automatic detailing grids.
11. Arabic SHX CAD Fonts: DXF text Unicode shapes can fail on older AutoCAD versions, requiring bound SHX/TrueType font packing.
12. 3D DXF Object Extrusion: DXF exports are flat 2D projection models; lacks true 3D solid element extrusion blocks.
13. Seismic Shear Wall Boundary Zones: Lacks specialized detailing templates representing high-confinement shear boundaries.
14. Post-Tensioned Slabs Tendons: PT tendon geometries and stressing profiles are not supported.
15. Footing slab-on-grade SOG Interfaces: Plinth beam SOG connection detail drawings are omitted.
16. Skewed Crossing Beams: Detailing assumes orthogonal crossings, producing approximated sectional cuts for skewed cross joints.
17. Custom Rebar Shape Library: Users cannot draw new rebar shapes; they must select from pre-defined BBS libraries.
18. Real-time Object snapping (OSNAP): Canvas lacks snapping points (endpoints, midpoints, perpendiculars) during interactive drafting.
19. Multiple Drawing Layout Tabs: Visual workspace is navigated via dropdowns rather than standard bottom sheet layout tabs.
20. Attribute Tag Block Binding: Text in DXFs uses clean standard MTEXT rather than bound attribute fields (ATTRIB).
